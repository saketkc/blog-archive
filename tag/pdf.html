Problem

Given \(f(x) = \frac{x}{2}\) for \(0 \leq x \leq 2\). Find the pdf of \(x_1+x_2\) for \(x_1,x_2\) which are i.i.d.

Wrong solution

$f*g(t) = \int f(w)g(t-w)dw $

Thus, blindly,

\(f_S(s) = \int_{0}^s \frac{x}{2}{s-x}{2} = \frac{s^3}{24}\) and \(0 \leq s \leq 4\)

But is \(f_S(s)\) a PDF? No.

\(\int_0^4 f_S(s)ds = \frac{4^4}{96} \neq 1\)

Respecting the bounds.

$f_S(s) = \int_{0}^s \frac{x}{2}{s-x}{2} $ for :

$$ 0 \leq s \leq 2 $$

and

$$ 0 \leq s-x \leq 2 $$

Which gives us, these bounds:

\(x-2 \leq w \leq x\) and \(0 \leq w \leq 2\)

In order to respect t-he above bounds,

\(f*g(s) = \int_{-\inf}^{\inf}f(x)f(s-x)dx\)

But, \(f(x) \neq 0\) for \(0 \leq x \leq 2\)

and \(f(x-w)\neq 0 $ for $x-2 \leq w \leq x\)

thus, \(f(x)f(x-w) \neq 0 $ for $max(0,x-2) \leq w \leq min(x,2)\)

And thus,

\(f*g(s) = \int_{max(x-2,0)}^{min(x,2)}f(x)f(s-x)dx\)
